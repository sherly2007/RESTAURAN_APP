## 📝 Descripción
Sistema de gestión de productos para un restaurante, desarrollado aplicando los principios de Programación Orientada a Objetos: herencia, encapsulación y polimorfismo. Permite registrar platos y bebidas, modificar precios de forma segura y mostrar el menú completo.
